# Proposal: Skills Manager

> 创建时间：2026-04-07
> 状态：待确认

## 背景

当前 `claw-skills/skills/` 目录下有 37 个技能，分散存放，缺乏可视化管理界面。用户需要一个统一的管理页面来查看、搜索、分类和启用/禁用这些技能。

## 目标

在 workbench 中新增 Skills Manager 工具模块，提供：

1. **分类浏览** — 按6大类别（开发工具、搜索、内容生成、AI/知识库、系统工具、智能体）展示技能
2. **搜索筛选** — 按名称/描述实时搜索
3. **详情查看** — 展示 SKILL.md 内容预览
4. **启用/禁用** — 切换技能状态
5. **来源标签** — 标注自研/开源/系统内置

## 约束

- 复用 workbench 现有登录认证（JWT）
- 后端遵循工具模块规范：`router.py` + `meta.json`
- 前端遵循 React + Tailwind CSS 风格
- 数据源：本地文件系统 `claw-skills/skills/`
- 数据库：SQLite（workbench 现有）

## 不做什么

- 不做技能编辑功能
- 不做技能删除功能
- 不做技能上传/安装功能
